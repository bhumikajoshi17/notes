📝 Paste App

A simple Paste App built with frontend technologies that allows users to create, view, and delete pastes — implementing basic CRUD operations (Create, Read, Update, Delete).

🚀 Features

✍️ Create a new paste

👀 View saved pastes

❌ Delete pastes easily

⚡ Responsive and clean UI

💾 Data handled using frontend logic (can be connected to backend later)

🛠️ Tech Stack

Frontend: React.js / HTML / CSS / JavaScript (based on your setup)

State Management: React Hooks / useState

Styling: Tailwind CSS / CSS Modules / Plain CSS (choose as per your project)

📦 Installation & Setup

Follow these steps to run the project locally:

# 1. Clone the repository
git clone https://github.com/your-username/paste-app.git

# 2. Navigate into the project directory
cd paste-app

# 3. Install dependencies
npm install

# 4. Start the development server
npm start


The app will run on http://localhost:3000
